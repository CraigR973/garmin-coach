# Codex prompt wrappers

OpenAI's current Codex custom-prompt docs describe invokable prompt files under
the user-level path `~/.codex/prompts`, not a shared project-level prompt path.
This repo keeps the thin wrapper sources here so they can be reviewed with the
project. To invoke them in Codex, symlink or copy these top-level `.md` files
into `~/.codex/prompts/` and restart Codex.

The canonical procedures remain in `/Users/craigrobinson/garmin-coach/docs/agent-commands/`.
